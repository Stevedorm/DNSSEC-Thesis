# DNSSEC-Thesis (LaTeX)

This repository contains my thesis manuscript written in LaTeX.

The pdf form of the write-up is in the _build directory

## Build (local)
Requirements:
- TeX Live (recommended) or MiKTeX
- latexmk

Clean - run this command before building to ensure new content is added:
```bash
./scripts/clean.sh
```
Build - make the pdf from the latex files:
```bash
./scripts/build.sh
```
